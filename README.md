# Cait Lorenzo-Mahon — Portfolio

Personal portfolio site for Cait Lorenzo-Mahon, Front End Developer and digital specialist based in Yorkshire, UK.

**Live site:** [clorenzomahon.github.io](https://clorenzomahon.github.io)

---

## About

A hand-crafted portfolio built from scratch with semantic HTML, modular CSS and vanilla JavaScript. Designed with accessibility, performance and thoughtful interaction in mind.

---

## Tech Stack

- HTML5 (semantic, accessible markup)
- CSS (custom properties, modular architecture)
- Vanilla JavaScript (no frameworks or dependencies)
- Google Fonts (Inter + Syne)

---

## Features

- Light / dark mode with system preference detection and localStorage persistence
- Scroll-triggered entrance animations with `prefers-reduced-motion` support
- Accessible mobile navigation with focus trap and keyboard support
- Experience timeline with animated accordion
- Fully responsive across all screen sizes
- WCAG 2.1 AA compliant focus styles

- Semantic HTML throughout
- Skip to main content link
- All interactive elements keyboard navigable
